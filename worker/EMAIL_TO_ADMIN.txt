WHAT I NEED INSTALLED (with sudo/admin access):
=============================================================================

1. DOCKER & DOCKER COMPOSE (Most Important!)
   - Docker Engine 20.10+
   - Docker Compose 2.0+
   - Installation: https://docs.docker.com/engine/install/
   
   Ubuntu/Debian quick install:
   curl -fsSL https://get.docker.com -o get-docker.sh
   sudo sh get-docker.sh
   sudo apt-get install -y docker-compose-plugin

2. OLLAMA (AI/LLM Engine)
   - Installation: curl -fsSL https://ollama.ai/install.sh | sh
   - Start service: sudo systemctl start ollama && sudo systemctl enable ollama
   - Download model: ollama pull llama3.1:latest
   - Note: This downloads 4.7GB, takes ~10 minutes

3. TESSERACT OCR (Document text recognition)
   - Ubuntu/Debian: sudo apt-get install tesseract-ocr tesseract-ocr-eng
   - CentOS/RHEL: sudo yum install tesseract tesseract-langpack-eng

4. POPPLER UTILS (PDF processing)
   - Ubuntu/Debian: sudo apt-get install poppler-utils
   - CentOS/RHEL: sudo yum install poppler-utils

5. POSTGRESQL CLIENT LIBRARIES (Database connectivity)
   - Ubuntu/Debian: sudo apt-get install libpq-dev
   - CentOS/RHEL: sudo yum install postgresql-devel

FIREWALL CONFIGURATION:
=============================================================================

Please open these ports:
- Port 8001 (Worker API - internal access only)
- Port 11434 (Ollama - localhost only)
- Port 5434 (PostgreSQL - if on same server)

Ubuntu/Debian:
sudo ufw allow 8001/tcp
sudo ufw allow 11434/tcp
sudo ufw allow 5434/tcp
sudo ufw reload

CentOS/RHEL:
sudo firewall-cmd --permanent --add-port=8001/tcp
sudo firewall-cmd --permanent --add-port=11434/tcp
sudo firewall-cmd --permanent --add-port=5434/tcp
sudo firewall-cmd --reload
