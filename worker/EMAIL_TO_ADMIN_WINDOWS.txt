=============================================================================
WHAT I NEED INSTALLED (with Administrator privileges):
=============================================================================

1. DOCKER DESKTOP FOR WINDOWS (Most Important!)
   Download: https://docs.docker.com/desktop/install/windows-install/
   - Install Docker Desktop Installer.exe
   - Choose "Use WSL 2" option during install
   - Verify: docker --version && docker compose version

2. OLLAMA (AI/LLM Engine) 
   Download: https://ollama.ai/download/windows
   - Install OllamaSetup.exe
   - After install, open PowerShell and run:
     ollama pull llama3.1:latest
   - This downloads 4.7GB AI model (~10-15 minutes)
   - Verify: ollama list (should show llama3.1:latest)

3. TESSERACT OCR (Document text recognition)
   Download: https://github.com/UB-Mannheim/tesseract/wiki
   - Install tesseract-ocr-w64-setup-5.3.x.exe (latest version)
   - Install to: C:\Program Files\Tesseract-OCR
   - ✅ IMPORTANT: Check "Add to PATH" during installation
   - Verify: tesseract --version

4. POPPLER UTILS (PDF processing)
   Download: https://github.com/oschwartz10612/poppler-windows/releases/latest
   - Download Release-xx.xx.x-0.zip
   - Extract to: C:\Program Files\poppler
   - Add to System PATH:
     PowerShell (as Admin):
     [Environment]::SetEnvironmentVariable("Path", $env:Path + ";C:\Program Files\poppler\Library\bin", [System.EnvironmentVariableTarget]::Machine)
   - Verify: pdfinfo -v (restart PowerShell first)

5. POSTGRESQL CLIENT (Optional but recommended)
   Download: https://www.postgresql.org/download/windows/
   - Install PostgreSQL (or just command line tools component)
   - This may already be installed if PostgreSQL is on the server

SPECIAL NOTES FOR WINDOWS:
=============================================================================

1. WSL 2 Requirement:
   - Docker Desktop requires WSL 2 (Windows Subsystem for Linux 2)
   - The Docker installer will set this up automatically
   - If issues occur, run: wsl --install && wsl --set-default-version 2

2. PATH Environment Variable:
   - Tesseract and Poppler must be added to System PATH
   - This allows the worker service to find these tools
   - After adding to PATH, restart PowerShell/Command Prompt

3. Administrator Privileges:
   - All installations require "Run as Administrator"
   - Firewall rules require Administrator PowerShell

4. Antivirus/Security:
   - May need to allow Docker, Ollama through Windows Defender
   - May need to whitelist the worker service ports
